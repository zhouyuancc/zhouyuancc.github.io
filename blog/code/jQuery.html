（一）通过选择器查找元素

     a) $(‘选择器’)

     b) jQuery支持绝大多数css选择器


    //获取标签

    console.log($('img'));//Object

    //CSS中所有的选择器

    //class="test1"

    console.log($('.test1').attr('src'));//img/img_02.jpg

 

（二）属性操作

     a) 获得属性    $(‘选择器’).attr(‘属性名’)

     b) 设置属性    $(‘选择器’).attr(‘属性名’, ‘属性值’)


    //改变标签中的内容

    $('img').attr('src','img/img_02.jpg');//全变了


（三）显示和隐藏

     a) 显示    $(‘选择器’).show()

     b) 隐藏    $(‘选择器’).hide() //没有留空位

     c) 显示和隐藏  $(‘选择器’).toggle() 或 $(‘img').slideToggle();//有动画效果


    //动画

    $('.btn1').click(function(){

        // 'linear' 线性的

//        $('img').toggle('linear');


        //带时间的动画

        $('img').toggle(2000);

    });

     b) 点击事件 （不常用）
